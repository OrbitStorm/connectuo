Requirements
------------

1. Microsoft Windows XP, Vista, Server 2008, Windows 7
2. Microsoft .Net 4.0
3. Internet Connection (Optional, this is required to view and update the Public Server list)

Support and Suggestions
-----------------------

If you need help with ConnectUO or want to make a suggestion please visit the ConnectUO forums and make a post in the appropriate forum

Support : http://www.runuo.com/community/forums/connectuo-support.156/
Suggestions : http://www.runuo.com/community/forums/connectuo-discussion.157/

Reporting a bug
---------------

If you think you have discovered a bug please goto http://www.runuo.com/bugs/ to post create a new bug in the issue tracker.  Please be sure to search the tracker for the bug to avoid duplicate bug postings.
